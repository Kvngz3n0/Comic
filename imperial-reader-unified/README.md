# Imperial Reader

**Black & Gold Manga Reader** — Desktop + Android, Offline Downloads, Library, Proxy Rotation, Cloudflare Bypass, EPUB/eBooks, CBZ Support, Mihon/Tachiyomi Backup Compatible.

## Quick Start

### Desktop
```bash
cd imperial-reader-unified
chmod +x build_scripts/*.sh
./build_scripts/build_desktop.sh
```

### Android
```bash
./build_scripts/install_android_studio.sh  # First time only
./build_scripts/build_android.sh
```

## Features

- Black & Gold Theme (#0A0A0A + #C9A227)
- Library with SQLite persistence
- Search History (50 entries)
- Offline Chapter Downloads
- 25 Built-in Proxies with Rotation
- Cloudflare Bypass (Mihon/Tachiyomi WebView Method)
- Full-screen Manga Reader
- EPUB eBook Reader
- Local CBZ File Support
- Mihon/Tachiyomi Backup Import/Export
- MangaDex Extension
- Auto-detect Desktop vs Android
